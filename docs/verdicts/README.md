# Fable Verdicts

Fable writes exactly one protocol verdict for each completed slice:

```txt
docs/verdicts/S-001.md
```

Required fields:

```md
| Field | Value |
| --- | --- |
| Slice | S-001 |
| Judge | Fable |
| Verdict | PASS / FAIL / PARTIAL |
```

Sol, Terra, and Luna may report evidence or reviewer findings, but they must not
write verdicts. The human owns the ship or stop decision after Fable judges.
